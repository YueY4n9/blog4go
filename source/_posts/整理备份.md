`range for 拷贝问题`

```go
func study() {
	list := []int{1, 2}
	for i := range list {
		list = append(list, i)
	}
	fmt.Println(list)
	for i := 0; i < len(list); i++ {
		list = append(list, i)
	}
	fmt.Println(list)
}
```

`切片翻转排序`

```go
func reverse(nums []int) []int {
	sort.Slice(nums, func(i, j int) bool {
		return nums[i] > nums[j]
	})
	return nums
}
```
